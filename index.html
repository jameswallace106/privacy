Privacy Policy

Last updated: April 2026

Overview

This application is a personal, private tool used solely to automate the posting of video content to TikTok. It is not a commercial product and is not distributed to any third parties.

Data Collection

This application does not collect, store, or share any personal data from users. No analytics, tracking, or logging of user information takes place.

TikTok API Usage

This application uses the TikTok API solely to upload and publish video content to a single, personally owned TikTok account. No data is read from or written to any other accounts.

Third Party Sharing

No data of any kind is shared with third parties. This application operates entirely on a local private machine.

Contact

For any questions regarding this privacy policy, please contact the developer directly.
